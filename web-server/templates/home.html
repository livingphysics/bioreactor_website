How It Works

  1. Upload: Submit your Python script through the upload form
  2. Queue: Your experiment is added to the queue (max 5 per user)
  3. Execute: When it's your turn, the experiment runs in an isolated container
  4. Monitor: Track progress and download results when complete

Features

  • Queue System: FIFO queue with one experiment running at a time
  • Admin Controls: Cancel, pause, resume, and reorder experiments
  • Real-time Status: Live updates on experiment progress
  • Result Download: Download experiment results as ZIP files
  • Session Management: Track your experiments across browser sessions
